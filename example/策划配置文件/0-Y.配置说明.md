#  Excel → JSON 配置导出工具说明

本工具用于将 Excel 配置表转换为 **服务端 / 客户端 JSON 数据**，支持通过 `config.toml` 灵活控制：

- 输出 JSON 文件名
- 是否生成服务端 / 客户端 / 双端数据
- 自动跳过未变更的配置表
- 支持多 Excel 配置表统一管理

---

## 一、Excel 表结构规范

Excel 至少需要 **4 行结构定义**：

| 行号 | 含义 | 示例 |
|----|----|----|
| 第 1 行 | 备注行 | 描述字段含义 |
| 第 2 行 | 字段标记 | `s` / `c` / `b` |
| 第 3 行 | 类型定义 | `int` / `string` / `json` |
| 第 4 行 | 字段名 | `id` / `name` / `value` |
| 第 5 行起 | 数据行 | 实际配置数据 |

### 1、字段标记说明（第 2 行）

| 标记 | 含义 |
|----|----|
| `s` | 仅服务端使用 |
| `c` | 仅客户端使用 |
| `b` | 服务端 & 客户端共用 |


### 2、字段类型说明（Excel 第 3 行）

Excel 第 3 行用于声明字段类型，工具会根据该类型将单元格字符串转换为对应的 JSON 数据类型。

**注意：我们在开发中统一使用int、string、json**

### 支持的类型一览表

| Excel 中写法 | 实际 JSON 类型 | 示例 Excel 值 | 生成的 JSON |
|------------|---------------|---------------|-------------|
| `int` | 整数（Number） | `100` | `100` |
| `integer` | 整数（Number） | `42` | `42` |
| `float` | 浮点数（Number） | `3.14` | `3.14` |
| `double` | 浮点数（Number） | `1.5` | `1.5` |
| `number` | 浮点数（Number） | `10` / `2.5` | `10` / `2.5` |
| `bool` | 布尔值（Boolean） | `true` / `false` | `true` / `false` |
| `boolean` | 布尔值（Boolean） | `true` / `false` | `true` / `false` |
| `string` | 字符串（String） | `abc` | `"abc"` |
| `str` | 字符串（String） | `hello` | `"hello"` |
| `text` | 字符串（String） | `任意文本` | `"任意文本"` |
| `json` | JSON 对象 / 数组 | `{"a":1}` | `{ "a": 1 }` |
| `array` | JSON 数组 | `[1,2,3]` | `[1,2,3]` |
| `object` | JSON 对象 | `{"x":10}` | `{ "x": 10 }` |

---

### 2️⃣ 空值规则（重要）

- **单元格内容为空或仅包含空格时**
- 无论声明什么类型，都会被转换为：null，而null默认不转出到JSON文件的字段中。
